    RandomPizza - Generate a random pizza suggestion¶ ↑

    home

    github.com/ilynne/RandomPizza

    Description¶ ↑

    RandomPizza does one very simple thing – generate a random pizza suggestion. RandomPizza is 100% vegan. Heart disease is the number one killer!

    Use¶ ↑

    Once installed, use the RandomPizza command to get a suggestion

    $ p = RandomPizza.new
